# Pokedex

É um projeto desenvolvido com intuito de exibir filtrar e listar pokemons

### Requisitos

- [Node](https://nodejs.org/en/)

### Iniciando

```
$ git clone https://github.com/viniciusleal34/pokedex.git
```

```
$ cd pokedex
```

```
$ npm install
```

```
$ npm start
```
